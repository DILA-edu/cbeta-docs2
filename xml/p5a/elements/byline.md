# byline

包含作品的主要責任陳述，出現在題名頁或作品開頭或結尾處。

## type="Author"

例 T01n0001.xml

```xml
<byline cb:type="Author">長安釋僧肇述</byline>
```

## type="Collector"

例 T04n0207.xml, p. 522b18

```xml
<byline cb:type="Collector">比丘道略集</byline>
```

## type="Editor"

例 X81n1571.xml, p. 402c05
