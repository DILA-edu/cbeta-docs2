## XML Comments

### CBETA todo type: a

未以 app 標記不同版本間的差異，例 T30n1581.xml

```xml
<lb n="0909b04" ed="T"/>施與，
<note resp="Taisho" n="0909035" place="foot text" type="orig">〔若他…薩〕二十字－【三】【宮】</note>
<note resp="CBETA" n="0909035" type="mod">
  <!--CBETA todo type: a-->（若他…薩）二十字【大】，〔－〕【宋】【元】【明】【宮】
</note>
若他所須，是名略說菩薩四種一切門
```
